이미지 저장 방법:

  1. 위의 이미지 영역에서 마우스 우클릭
  2. "다른 이름으로 저장" 또는 스크린샷 도구 사용
  3. 파일명을 og-image.png로 저장
  4. 웹사이트 폴더에 저장하면 자동으로 연결됩니다

권장 도구:

  • Windows: Snipping Tool 또는 Alt + Print Screen
  • Mac: Cmd + Shift + 4
  • 브라우저: 개발자 도구 → 스크린샷 기능

사용 효과:

  • 카카오톡, 페이스북 등에서 링크 공유시 이미지 표시
  • Twitter 카드로 트윗에 이미지 미리보기
  • 검색 엔진에서 더 나은 노출 효과
